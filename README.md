# Paco Romane Comedy Show Updates

A reusable, responsive HTML email and web page for Paco Romane comedy shows, ticket links, featured content, and giveaways.

Live page:

https://pacoromane.github.io/comedy-show-updates/

## Main files

- `index.html` — current published email
- `index-backup.html` — backup of the previous version
- `assets/images/` — masthead, social icons, photos, and other hosted images

## Updating the email

1. Edit `index.html`.
2. Update the preheader text.
3. Update the Paco note.
4. Update the featured show.
5. Add, remove, or edit complete show-card blocks.
6. Update featured content and giveaway links.
7. Commit changes.
8. Wait about one minute.
9. Test the live page on desktop and mobile.
10. Send a test email before sending to the full list.

## Image URLs

Images in:

`assets/images/filename.png`

are publicly available at:

`https://pacoromane.github.io/comedy-show-updates/assets/images/filename.png`

Use lowercase, descriptive filenames without spaces.

Examples:

- `masthead.png`
- `loveless-july-2026.png`
- `youtube-icon.png`

## Important editing rule

For upcoming shows, duplicate or delete an entire block between:

`SHOW CARD START`

and:

`SHOW CARD END`

Do not copy only part of a table.

## Email testing

Test before sending in:

- Gmail desktop
- Gmail mobile
- Apple Mail
- iPhone Safari
- Outlook, when possible

## Required footer information

Before sending a bulk promotional email, replace the mailing-address placeholder with a valid business mailing address and provide a working unsubscribe method.
